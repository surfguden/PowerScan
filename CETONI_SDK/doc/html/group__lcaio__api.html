labbCAN Analog I/O API

Overview

Analog I/O specific functions.

This module defines all labbCAN analog I/O specific functions, identifiers and data types for controlloing analog I/O modules attached to the labbCAN bus. Analog modules are used for measurement of analog input signals or for generation of analog output signals. This module offers a device independent API for access to all supported labbCAN analog devices.
